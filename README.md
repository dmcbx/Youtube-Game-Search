# Youtube-Game-Search
Chromium based extension that auto-detects game names from any site for a smart YouTube search.

## ‚ú® Core Features

* **Smart Game Detection:** Right-click any game **image** or **title** on any site. The extension intelligently finds the game's name.

## üõ†Ô∏è How to Install (Manual Guide)

This is an unpacked developer extension. Manual installation takes only 1 minute.

1.  **Download:** Go to the **[Releases Page](./releases)** and download the latest `.zip` file.
2.  **Unzip:** Extract the `.zip` file into a permanent folder on your computer (e.g., in your "Documents" folder).
3.  **Open Browser:** Open Chrome (or Edge).
4.  **Open Extensions Page:** Type `chrome://extensions` (or `edge://extensions` for Edge) in your address bar and press Enter.
5.  **Enable Developer Mode:** In the top-right corner, click the **"Developer mode"** toggle to turn it **ON**.
6.  **Load the Extension:** Click the **"Load unpacked"** button.
7.  **Select Folder:** In the window that opens, select the folder you unzipped in Step 2.

**Done!** The extension is now active. Right-click any game to test it.
